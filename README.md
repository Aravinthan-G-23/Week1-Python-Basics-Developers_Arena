# Personal Introduction Program

## Project Overview
This project is a beginner-friendly Python application that collects user information and displays a personalized welcome message.

## Objectives
- Learn Python variables.
- Understand input() and print().
- Practice f-strings.
- Build an interactive console application.

## Setup Instructions

### Step 1
Install Python 3.x.

### Step 2
Clone the repository:

```bash
git clone https://github.com/username/Week1-Python-Basics.git
```

### Step 3

Run the program:

```bash
python personal_intro.py
```
